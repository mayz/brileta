# Catley

A post-apocalyptic roguelike.

## Getting Started

1. Install **Python 3.13**.
2. Run `uv sync` to create a virtual environment and install dependencies from `uv.lock`.
3. Install the project in editable mode:
   ```sh
   uv pip install -e .
   ```
4. Execute the test suite:
   ```sh
   pytest -q
   ```
